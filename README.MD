# Project Citadel

## Настройка сборки

##Django
```bash
$ cd backend/citadel
$ pip install -r ./requirements.txt
$ python manage.py migrate
$ python manage.py runserver 8002
```

##Vue
```bash
$ cd frontend
$ npm install
$ npm run serve
```
